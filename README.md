\# BulkyBook.AspNetCore



ASP.NET Core MVC web application with MS SQL Server backend.



\## Technologies

\- C#

\- ASP.NET Core

\- Entity Framework Core

\- MS SQL Server



\## Features

\- CRUD operations: Create, Read, Update, Delete

\- MVC architecture: Controllers, Services, Repositories

\- Database interaction using Entity Framework Core



\## Description

This web application demonstrates backend development and database integration using ASP.NET Core. 

It was created to practice structured architecture, REST-like endpoints, and handling business logic.
